SpecialTheater 앱은?

영화를 좋아하는 사람들을 위한 앱으로서, 해당 영화를
어느 상영관에서 보았을 때 가장 좋았는지를 알려주는 용도로 만든 앱입니다.
다른 사람들과 특별한 상영관에서의 경험을 공유하는 것으로
영화를 더 쾌적한 장소에서 관람할 수 있다.

SpecialTheater 기능은?


  • 검색기능

    우선, 타 영화예매 사이트처럼 현재 상영중인 영화를 검색 할 수 있다.


  • 상영중인 특별관 목록

    선택한 영화가 어떤 특별관에서 상영 중인지 한 눈에 보여준다.


  • 리뷰 목록

    다른 사용자가 해당 영화를 선택한 상영관에서 관람했을 때의 리뷰 목록들과 선택한 상영관의 전체 리뷰를 한 눈에 보여준다.


  • 리뷰 작성

    자신이 경험한 상영관의 리뷰를 작성할 수 있다.

추가 기능 소개

위 주요 기능들을 제외하고도 SpecialTheater는 아래와 같은 추가 기능들을 지원합니다.


  • 리뷰 신고기능

    사용자는 다른 사용자의 부적절한 리뷰들에 대해 신고할 수 있습니다.


  • 리뷰 삭제기능

    사용자는 자신이 작성한 리뷰를 삭제할 수 있습니다.


  • 다크모드

    SpecialTheater는 다크모드를 지원합니다.
